User Guide

Introduction

Welcome to Event Finder, an easy way to register and explore events in your area. This guide will take you through the key features of Event Finder.

Getting Started

  • How to Create an Account:
    1. Go to the Register page and enter your details.
    2. Click Register to confirm your registration.
  • How to Log In:
    1. Enter your username and password on the Login page.
    2. Click Login to access your personalized Event Finder account.
  • How to Log Out:
    1. If you are logged in, click Log out in the upper-right corner of the screen.
    2. On the next screen, click Confirm to log out.

Feature Walkthroughs

  • How to Register an Event:
    1. Click User Portal on the menu.
    2. Under Your Events, select Event Registration.
    3. Fill out the event details, ensuring the address is accurate.
    4. Click Register Event to confirm.
  • How to Edit an Event:
    1. Go to Your Events in the User Portal section.
    2. Select the event you want to edit.
    3. Click the Update Event button.
    4. Enter the updated details and click Submit.

    Note: You can only edit events you have created.

  • How to Delete an Event:
    1. Go to Your Events in the User Portal.
    2. Select the event you want to delete.
    3. Click the Delete Event button.
    4. Click Confirm to finalize the deletion.

    Note: You can only delete events you have created.

  • How to Bookmark an Event:
    1. Browse events on the homepage.
    2. Select the event you want to bookmark.
    3. Click the Bookmark button to save it to your list.
  • How to Delete a Bookmark:
    1. Go to the Bookmarks section.
    2. Browse your bookmarks and click Delete Bookmark.
    3. On the next page, click Confirm to delete the bookmark.
  • How to Update Your Information:
    1. Go to the User Portal.
    2. Select Update User Information.
    3. Enter your updated details and click Submit.

Troubleshooting & FAQs

  • Can I undo an event deletion?

    • No, event deletions are final. You will need to register the event again if you want to add it back.

  • How can I update my password?
    1. Go to User Portal.
    2. Select Update User Information.
    3. Enter your new password and confirm it.
    4. Click Submit to save the new password.
    5. Log out, then use your new password to log back in.
  • Can I browse events without an account?

    • Yes, you can browse events without logging in. However, to bookmark or create events, you need to be logged in.
